# Geometry Wars

## Usage
```
$ git clone https://github.com/vssense/GeometryWars.git
$ mkdir build
$ cd build
$ cmake ..
$ make
$ ./game
```
- Use Up, Down, Left and Right keys to move player, mouse to shoot.
